# Merge, Rebases, conflitos e resoluções

## Rebase

O rebase é uma alternativa ao merge para integrar mudanças de uma branch para outra. Ele reaplica os commits de uma branch em cima de outra, criando um histórico linear.

### `git rebase`

_"Reapply commits on top of another base tip"_
